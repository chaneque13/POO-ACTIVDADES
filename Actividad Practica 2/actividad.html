JavaScript Externo y interno

JavaScript es un lenguaje de programación interpretado, dinámico y de alto nivel, fundamental para el desarrollo web moderno. Aunque comparte similitudes sintácticas con Java, no está relacionado con este lenguaje y se utiliza principalmente para añadir interactividad a páginas web, permitiendo cambios dinámicos en el contenido, control de multimedia, animaciones y gestión de eventos en tiempo real.
